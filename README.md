# Physics of Life, Data, and Epidemiology

This repository contains the course materials, laboratory exercises, and final project developed for the Physics of Life, Data, and Epidemiology course.

The content applies methods from physics and data analysis to biological and epidemiological systems, with an emphasis on quantitative modeling, data-driven approaches, and statistical analysis.

Contents include:
* Lecture and reference materials
* Laboratory notebooks and scripts with data analysis and modeling exercises
* Final project and related resources
